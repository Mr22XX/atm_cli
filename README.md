# Simulasi ATM CLI dengan Node.js & MySQL

## Fitur
- Registrasi akun
- Login akun
- Cek saldo
- Setor dan tarik tunai
- Transfer antar akun
- Riwayat transaksi disimpan

## Teknologi
- Node.js
- Commander.js
- MySQL
- Inquirer

## Menjalankan Aplikasi
1. Clone repo
2. `npm install`
3. Jalankan perintah:
   - `node app.js register`
   - `node app.js login`
   - `node app.js check-balance` dll.
